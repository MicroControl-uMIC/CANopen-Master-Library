CANopen FD Master Library

CANopen® is an open layer-7 protocol based on CAN (Controller Area Network). CANopen is mainly applied in machines as embedded network system. Moreover, it is used as a general industrial communication system with a data transfer rate of up to 1 Mbit/s.

The CANopen FD Master Library supports the following services:

  • Service Data Objects (SDO service, Client & Server)
  • Process Data Objects (PDO service)
  • Network Management (NMT service)
  • Emergency Messages (EMCY service)
  • Synchronisation Messages (SYNC Service)
  • Layer Setting Services (LSS service, Master)

API

The CANopen FD Master Library provides a C interface and a C++ interface class for the Qt library. Please refer to CANopen FD Master API overview for details.
