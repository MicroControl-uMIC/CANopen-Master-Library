QCoObject Class Reference

Detailed Description

This class is the basic representation of an CANopen object, i.e. an entry inside a CANopen object dictionary.

Inherits QObject.

◆ sizeLimit()

quint32 QCoObject::sizeLimit ( void  )

Return the maximum number of data bytes that can be stored in this object. The value is determined by the data type. For basic data types (INTEGER8, INTEGER16, etc) the function returns the size of the data type (i.e. 1 for INTEGER8). For data types with a variable size the function returns a limit value. Use setSizeLimit() to configure the value individually.
