QCoScriptEdit Class Reference

Detailed Description

This class is based on the QTextEdit widget and allows entering of JavaScript commands as well as displaying the result of JavaScript commands.

The script browser provides the functionality of the JavaScript console standard (see https://github.com/whatwg/console). For supporting the console standard the QCoScriptBrowser class must be attached to the script engine class (QCoScript) via the attachScriptEngine() method, using the object name "console".

Inherits QTextEdit.

Parameters
clObjectNameV- Name of JavaScript object
pclEngineV- Pointer to script engine

The method is used to enable access to scripting functions of this class. The pointer pclEngineV defines the script engine. The parameter clObjectNameV defines the name of the JavaScript object.

For support of the JavaScript console standard it is recommended to use the name "console" for clObjectNameV.
